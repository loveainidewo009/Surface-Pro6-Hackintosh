本项目提供完整的 Surface Pro 6 黑苹果 EFI 配置与详细教程，包含：

- **EFI 文件夹**：OpenCore 1.0.7 引导配置 + 全套驱动

## 使用必读

⚠️ **当前 EFI 中的序列号为占位符（FAKESERIAL / FAKEMLB / 00000000-0000-0000-0000-000000000000），必须替换为有效的真实三码，否则无法正常使用 Apple 服务。**

⚠️ **Surface 安全启动（Secure Boot）支持仍在开发中，请保持关闭。**

⚠️ **摄像头无解，内置 Wi-Fi/蓝牙无解，必须外接 USB 网卡。**
